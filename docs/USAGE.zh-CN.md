# 使用手册

[English](USAGE.md) | [简体中文](USAGE.zh-CN.md)

本指南面向最终用户，介绍如何运行 `GoldPrice`、切换报价源以及添加桌面小组件。

## 1. 启动应用

### 通过 Xcode 运行

1. 打开 `GoldPrice.xcodeproj`
2. 选择 `GoldPrice` scheme
3. 为 `GoldPrice` 和 `GoldPriceWidgetExtension` 分配签名 Team
4. 在 `My Mac` 上运行

### 通过 `.dmg` 安装运行

1. 双击 `GoldPrice.dmg`
2. 将 `GoldPrice.app` 拖入 `应用程序` 文件夹
3. 如果 macOS 阻止首次启动，请在 `系统设置 > 隐私与安全性` 中允许

## 2. 菜单栏面板

启动后，应用入口位于菜单栏。

点击金价数字即可打开面板。面板通常显示：

- 当前 `USD / OZ` 价格
- 当前 `RMB / g` 价格
- 上次更新时间
- 当前会话涨跌
- 数据源切换按钮
- 币种切换按钮（在 `$/OZ` 和 `¥/G` 之间切换菜单栏小字）
- `详情`、`刷新` 和 `退出` 按钮

## 3. 数据源切换

菜单栏面板支持三种模式：

- `自动`
  - 默认模式
  - 优先使用 `Kitco`
  - 失败时自动回退到 `Gold API`
- `Kitco`
  - 仅使用 Kitco
  - 如果 Kitco 页面变动或不可达，将直接报错
- `Gold API`
  - 仅使用 `api.gold-api.com`

切换数据源后：

- 当前历史缓冲区会清空并重新采样
- 应用立即发送新请求
- 你的选择会在本次运行期间保留

## 4. 币种切换

菜单栏面板提供币种切换按钮，点击可在美元和人民币显示之间切换：

- `$` 模式：菜单栏显示美元/盎司
- `¥` 模式：菜单栏显示人民币/克

你的选择会自动保存，下次启动时仍然有效。

## 5. 价格说明

应用主要关注两个价格：

- `USD / OZ`
  - 每盎司美元报价
- `RMB / g`
  - 换算后的每克人民币价格

如果人民币价格显示为 `--`，说明当前请求未能获取到有效的 `USD/CNY` 汇率。

## 6. 详情窗口

在菜单栏面板中点击 `详情` 打开主窗口。

详情窗口提供：

- 更大的主要报价展示
- 短期价格走势图
- 当前会话涨跌
- 当前数据源和上次更新时间

主应用每秒刷新一次，因此详情窗口比小组件更实时。走势图仅在报价源返回新数据时推进。

## 7. 添加桌面小组件

添加小组件前，请确保主应用已成功启动过至少一次。

### 方法 A：桌面右键菜单

1. 在 macOS 桌面空白处右键
2. 选择 `编辑小组件`
3. 搜索 `GoldPrice` 或 `国际金价`
4. 选择 `小号` 或 `中号`
5. 添加即可

### 方法 B：通知中心

1. 打开通知中心
2. 滚动到底部
3. 点击 `编辑小组件`
4. 搜索并添加 `GoldPrice`

## 8. 小组件刷新

小组件不支持秒级实时更新。

当前行为：

- 正常情况下约每 15 分钟刷新一次
- 请求失败后约 5 分钟重试

这属于 `WidgetKit` 的正常限制，并非应用 bug。

## 9. 常见问题

### Q：为什么 `Kitco` 模式会失败？

可能的原因：

- Kitco 页面结构发生了变化
- 网络访问失败
- 发生了临时超时

尝试切换回 `自动` 或直接选择 `Gold API`。

### Q：为什么看不到小组件？

检查以下内容：

- 主应用已至少启动过一次
- `GoldPriceWidgetExtension` 签名成功
- 两个 target 使用相同的 Team
- bundle identifier 没有冲突

### Q：为什么价格没有更新？

检查以下内容：

- 网络连接是否正常
- 当前数据源是否可达
- 是否过于频繁地切换数据源

你也可以在菜单栏面板中点击 `刷新` 立即获取最新数据。

### Q：为什么不能直接把 app 发给别人？

无签名的本地构建通常只能在你的 Mac 上运行。要分发给他人，你还需要：

- Developer ID 签名
- 公证（notarization）
- 装订（stapling）

详见[构建与发布](BUILD_AND_RELEASE.md)。
