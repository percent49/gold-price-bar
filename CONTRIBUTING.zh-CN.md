# 贡献指南

[English](CONTRIBUTING.md) | [简体中文](CONTRIBUTING.zh-CN.md)

欢迎为 `GoldPrice` 做出贡献。

这是一个紧凑的原生 macOS SwiftUI 应用，请牢记以下原则：

- 优先使用原生实现，避免不必要的依赖
- 菜单栏体验优先于功能膨胀
- 保持数据展示简洁易读
- 保持文档与行为一致

## 开始之前

1. 安装 Xcode 15 或更高版本
2. 打开 `GoldPrice.xcodeproj`
3. 配置本地签名 Team
4. 先确认 `Debug` 构建可以正常工作

## 适合贡献的方向

- 报价源可靠性
- 解析器健壮性
- UI 打磨
- 小组件展示
- 文档更新
- 构建和发布改进

## PR 前检查清单

请至少验证以下内容：

- `Debug` 构建成功
- 相关文档已更新
- 菜单栏面板无明显布局退化
- 如果修改了报价获取逻辑，请验证 `自动` 模式和至少一个固定数据源
- 如果修改了小组件，请确保 extension 仍然能构建

## Pull Request 说明

一个好的 PR 描述应包含：

- 改动的目的
- 主要实现要点
- 手动验证步骤
- 已知限制或遗留问题

如果改动影响 UI，请尽可能附带截图。

## 文档说明

本仓库同时维护英文和简体中文文档。当你修改行为时，如果改动影响用户或贡献者，请同时更新两边的文档。

同步文档的时机：

- 数据源行为变更
- 刷新策略变更
- 小组件限制变更
- 安装或打包流程变更
- 分发步骤变更

## 许可证

尚未选择许可证。在贡献代码之前请留意这一点。
